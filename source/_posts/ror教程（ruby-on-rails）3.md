Hash.new 可接受一个参数指明元素的默认值，即当键不存在时返回的值
```
>> h = Hash.new
=> {}
>> h[:foo]            # 试图获取不存在的键 :foo 对应的值
=> nil
>> h = Hash.new(0)    # 让不存在的键返回 0 而不是 nil
=> {}
>> h[:foo]
=> 0
```
在类中调用方法或访问属性时可以不用 self.（赋值例外）
